**拆解日期**: 2026-06-28

## 一、项目概述

这是一个 **AI驱动的竞赛策划书自动生成系统**，面向参加"互联网+"、"挑战杯"、"节能减排"等大学生竞赛的团队。用户输入项目资料，系统调用 LLM（Claude / DeepSeek）自动生成国奖级别的完整策划书，包含正文、图表、PPT、答辩预演、查重报告等。

| 维度 | 详情 |
|------|------|
| 语言 | Python 3 |
| AI引擎 | Claude (Anthropic) / DeepSeek 双引擎 |
| 界面 | Streamlit Web应用 + CLI命令行 |
| 输出格式 | Word(.docx) / PDF / HTML / Markdown / PPT(.pptx) |
| 支持赛事 | 8个主流大学生竞赛 |
| 模板数量 | 8个赛事模板 + 6种视觉风格 |
| 知识库 | 修辞库(9类) + 图表组件库 + 视觉风格库 |

## 四、知识库体系（双库驱动）

### 4.1 国奖模板库 (templates/)
每个模板定义：
- 章节结构（8章标准：执行摘要/背景/技术/产品/市场/团队/财务/未来规划）
- 页数配比（技术25% > 市场20% > 背景15% = 产品15%）
- 字数上限（技术5500字 > 市场4500字 > 背景4000字）
- 视觉风格默认值
- 图表要求清单

### 4.2 修辞库 (rhetoric_lib/)
按章节分类的国奖级表达模板：
- 常用句式 → 替换用户原始表达的占位符
- 数据描述模板 → 将原始数据转化为学术表达
- 过渡句 → 章节间衔接

### 4.3 视觉风格库 (visual_styles/)
6种配色方案，每种定义：
- 主色/辅色/强调色（HEX色值）
- 字体族（标题/正文/代码）
- 图表配色（柱状图/饼图/折线图）
- 封面布局参数

## 五、输入处理：5类资料分级响应

系统将用户输入分为5类：

| 类别 | 内容 | 必填 |
|------|------|------|
| competition_info | 赛事组别/赛道/类别 | ✅ |
| project_material | 项目名称/简介/技术原理/创新点/市场数据 | ✅ |
| team_info | 负责人/成员/导师/过往获奖 | ✅ |
| doc_requirement | 目标页数/配色方案 | 可选 |
| evidence | 专利/软著/检测报告/合作协议 | 可选 |

完整度分级响应：
- **≥80%** → 直接执行全流程
- **50-80%** → 引导补全缺失资料
- **<50%** → 终止并输出补充指引

## 八、两种使用模式

### 模式一：Web界面 (Streamlit)
```bash
streamlit run app.py
```
- 精美的Web表单界面
- 支持JSON/Word/TXT文件导入自动填充
- 实时进度条显示生成状态
- 分类下载文件
- 演示模式（一键体验）

### 模式二：命令行
```bash
python main.py                  # 交互式CLI引导
python main.py --demo           # 运行演示案例
python main.py --config a.json  # 从JSON配置加载
```

## 九、设计亮点

1. **双库驱动** — 国奖模板库 + 修辞库，确保输出质量
2. **分级响应** — 根据资料完整度智能调整流程
3. **零虚构原则** — 不从知识库编造数据，严格基于用户输入
4. **双AI引擎** — Claude + DeepSeek 互为备份
5. **10步流水线** — 从输入到输出的完整自动化
6. **答辩一体化** — 策划书+PPT+答辩预演+路演稿全部自动生成
7. **6种视觉风格** — 适配不同赛事主题
8. **资料隐私** — 7天自动清除客户数据
9. **文件导入** — 支持JSON/Word/TXT拖拽上传自动填充
